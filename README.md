# TRMNL Daily Strip

A TRMNL recipe that displays daily comic strips from around the web on your e-ink display.

## Supported Comics
- Agnes
- Broom Hilda
- Dick Tracy
- Doonesbury
- (more coming soon)

## Issues & Support
If something isn't working, please [open an issue](../../issues) and I'll take a look.

## How it works
This recipe fetches daily comic strips from public comic websites via a dedicated server 
and displays them on your TRMNL device. Comics are manually curated and selected for optimal
viewing on e-ink displays — strips with non-standard aspect ratios, low image resolution, or 
poor contrast are excluded from the list.
